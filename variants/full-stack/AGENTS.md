# Agent instructions

This is a Next.js project. When you need Next.js documentation, read the
version-matched docs bundled with the installed Next.js, in
`node_modules/next/dist/docs/`. They match the exact Next.js version this project
uses, so they are more reliable than general knowledge, which can be out of date
for this version.

Prefer those bundled docs over memory for anything about the App Router, routing,
caching, server and client components, data fetching, and configuration. If the
bundled docs and your prior knowledge disagree, the bundled docs win.

For other libraries here, prefer the installed version's own documentation over
general knowledge when the two might differ.
